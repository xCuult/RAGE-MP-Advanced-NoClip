# NoClip which allows the player or the vehicle in which the player is sitting to fly
# Flexible configuration available

#Controls
| Key | Action |
| --- | --- |
| F2 | Toggle NoClip |
| W | Move forward |
| S | Move backward |
| A | Move left |
| D | Move right |
| LMB | Increase speed while flying |
| RMB | Decrease speed while flying |
